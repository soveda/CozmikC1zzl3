Presets

Drag points on either graph to edit stages, or type exact values below. Factory presets turn into a copy the first time you change them. Bounce is the reset preset.
Selected lane: only the active window can be moved. Point numbers: when stages stack, only the highest number is shown. Dragging: move a point to change its level and timing together. Preset status: Local only is stored in this browser; Saved is on the card; Changed has edits not yet saved to the card. Play / Stop: Play previews the envelope in the browser on a loop. Stop ends the browser preview only.

Pitch Envelopes

Pitch points are capped to the amplitude/PD envelope length. Pitch 1 drives oscillator 1 and Pitch 2 drives oscillator 2 with the full dual-oscillator test firmware.

Temporary changes: Load RAM, Load Sound Preset, and Send Settings are lost when the card restarts. Save Envelope Only writes the envelope to the selected card slot. Save Sound Preset stores the envelope, slot name, and current settings together on Rad/Gnarly; on Core firmware it safely falls back to a collapsed Core-compatible envelope save. Read Envelopes from Card also restores the saved settings for each sound preset when the card supports them. To make the global hardware startup baseline, move the switch from middle to down and hold it until the card confirms the save.
Version compatibility: This lab can talk to C1ZZL3 Core, Rad, and future Gnarly-style firmware. The editor auto-checks capability before sound-preset saves when needed; Read Settings from Card and Read Envelopes from Card can also refresh the displayed capability. Older Core firmware receives a collapsed Amp/PD/Pitch payload; Rad/Gnarly firmware receives the full dual-oscillator lanes and sound-preset data.
