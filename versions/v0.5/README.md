# C1ZZL3 v0.5 Snapshot

This folder contains a separate v0.5 firmware snapshot restored from:

`backups/promote_detune_fixed_level_20260611_232346/`

Files:

- `C1ZZL3_v0.5.cpp`
- `C1ZZL3_web_midi_v0.5.uf2`

This is the hardware-tested build before the fixed-level oscillator 2 detune
change. In this version, held-switch Main controls both oscillator 2 detune and
oscillator 2 level from centre.

It still includes the earlier confirmed Web MIDI, USB MIDI, mode-specific knob
pickup, matched CV In 1 / CV In 2 routing, and Turing background CV behaviour.

The root production files are not changed by this snapshot.
